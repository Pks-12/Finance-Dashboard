
Finance Dashboard UI

Overview
The Finance Dashboard is a responsive web application that helps user track their income, expenses and overall finance insights in a simple and visual way.
It Provides charts, transaction management, and filtering features to better understood spending patterns.

Features
·	Interactive Dashboard with charts (Bar & Pie).
·	Track Income and Expenses.
·	Search and Filter Transactions.
·	Sort transactions by date or amount.
·	Financial Insights visualization.
·	I try Responsive Design.

Tech Stack
·	React.js
·	CSS
·	Recharts (for data visualization)
·	React-icons




Tools
·	Vite
·	Git & GitHub

How It Works
·	Transactions data is stored in a local JSON file. file name is transactions.js
·	Users can:
·	View all transactions
·	Filter by Income/Expense
·	Search transactions
·	Sort data oldest and newest
·	User can see the Recent Transactions.
·	If You are views only data can see, if you switch the admin from viewers the you add the transactions or edit the transactions or delete the transactions.
·	Charts are generated using transaction data for better insights.
·	User can see Highest Spending category click buttons insights
·	User can monthly compare using charts in Insights.
